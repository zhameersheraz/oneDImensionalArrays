# AverageOfFloatingPointNumbers

Reads 10 doubles from standard input and prints their arithmetic mean to two decimal places.

## Compile & Run

```bash
javac AverageOfFloatingPointNumbers.java
java AverageOfFloatingPointNumbers
```
